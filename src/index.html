Keyboard Controls

    Clock

  • T: Start/Stop Game Clock
  • Spacebar: Start/Stop Shot Clock
  • R: Reset Shot Clock (12s)
  • Shift + R: Reset Shot Clock (14s - O.Rebound)

    • Score

  • Arrow Up: Home Score +1
  • Arrow Down: Home Score -1
  • Arrow Right: Away Score +1
  • Arrow Left: Away Score -1

    • Fouls

  • F: Home Fouls +1
  • Shift + F: Home Fouls -1
  • J: Away Fouls +1
  • Shift + J: Away Fouls -1

    • Timeouts

  • T: Home Timeouts -1
  • Shift + T: Home Timeouts +1
  • Y: Away Timeouts -1
  • Shift + Y: Away Timeouts +1
  • Enter: Set Custom Game Time (prompts)
  • H: Show/Hide This Help
