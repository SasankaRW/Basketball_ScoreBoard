Keyboard Controls

    Clock

  • T: Start/Stop Game Clock
  • Spacebar/Right click: Start/Stop Shot Clock
  • G: Reset Game Clock
  • R/Left click: Reset Shot Clock (12s)

    • Score

  • Arrow Up: Home Score +1
  • Arrow Down: Home Score -1
  • Arrow Right: Away Score +1
  • Arrow Left: Away Score -1

    • Fouls

  • F: Home Fouls +1
  • Shift + F: Home Fouls -1
  • J: Away Fouls +1
  • Shift + J: Away Fouls -1
  • Shift + C: Set Custom Shot Clock (prompts for seconds)
  • Enter: Set Custom Game Time (prompts)
  • H: Show/Hide This Help
